# Mangalam HDPE Pipes Product Landing Page

A fully responsive product landing page built using **Vanilla HTML, CSS, and JavaScript** based on the provided Figma design.

## 🚀 Features

* Fully responsive design
* Sticky responsive header
* Product image slider/carousel
* Smooth hover animations
* FAQ accordion
* Industry slider section
* Popup modal for quote requests
* Technical specification table
* Modern UI matching Figma design
* Semantic HTML5 structure
* Clean modular CSS
* Vanilla JavaScript interactions

---


# 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts (Inter)

---

# 📱 Responsive Design

The website is optimized for:

* Desktop
* Laptop
* Tablet
* Mobile devices

All sections use a maximum width of:

```css
max-width: 1240px;
```

to maintain layout consistency on larger screens.
